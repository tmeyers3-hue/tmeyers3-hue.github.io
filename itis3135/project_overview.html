Project Overview:

  • This application is a teaching tool specifically for CSS Grid
  • The intended users for this site are novice web developers who want help with CSS grid
  • The Site will include cheatsheets, study tools and quizzes to help cement grid concepts

Site Map:

  • Home
    • About

      Displays about statement and information, viewable by all users, no interaction

    • Contact

      Displays contact information for the developer, viewable by all users, able to "send a message"

  • Cheatsheets
    • Basic Grid Concepts

      A list of basic grid components and techniques, viewable by all users, little interaction (maybe some spoiler answers)

    • Advanced Grid Concepts

      A list of more advance grid components and techniques, viewable by all users, little interaction (maybe some spoiler answers)

  • Study Tools
    • Flashcards

      Flashcards for studying confusing topics, viewable by all users, interaction for flipping the cards. Will have buttons and fields for user input

    • Examples

      Example code, fill in the blank for interaction. Viewable by all users

  • Quizzes
    • Beginner Level

      Beginner MCQ and FRQ for Grid Concepts. Scored at the end. Users will be entering answers. Will have buttons and fields for user input

    • Advanced Level

      Advanced MCQ and FRQ for Grid Concepts. Scored at the end. Users will be entering answers. Will have buttons and fields for user input
